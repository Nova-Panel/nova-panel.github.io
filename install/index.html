راهنمای نصب و پیکربندی

آموزش گام‌به‌گام راه‌اندازی پنل Nova Proxy روی Cloudflare Workers

۱. اتصال KV Namespace

برای ذخیره تنظیمات، لاگ‌ها و داده‌های پنل، باید KV Storage ایجاد و به worker متصل کنی:

  1. در Cloudflare Dashboard برو به Workers & Pages → KV
  2. روی Create Namespace بزن و اسمش رو بذار مثلاً NOVA_KV
  3. برو به صفحه worker خودت ← Settings → Variables
  4. بخش KV Namespace Bindings ← Add binding
  5. Variable name: KV — KV namespace: اسمی که ساختی (مثلاً NOVA_KV)
  6. روی Save بزن

۲. متغیرهای محیطی (Environment Variables)

برای پیکربندی worker، برو به Settings → Variables → Environment Variables و متغیرهای زیر رو اضافه کن:

متغیرالزامیتوضیح
ADMIN ضروری رمز عبور پنل مدیریت. هر چی دلت میخواد بذار.
KEY پیشنهادی کلید رمزنگاری برای تولید UUID. امنیت بهتره حتماً یه مقدار دلخواه بدی.
UUID اختیاری UUID دلخواه (فرمت:xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx). اگر ندی از ADMIN+KEY ساخته میشه.
HOST اختیاری هاست مجاز (چندتا رو با کاما یا خط جدید جدا کن). اگه ندی، از هاست درخواست استفاده میشه.
PROXYIP اختیاری آی‌پی‌های پروکسی دلخواه. اگر ندی، بر اساس colo کلودفلر ساخته میشه.
GO2SOCKS5 اختیاری لیست سفید دامنه برای SOCKS5. پیش‌فرض داره (tapecontent.net, ...).
PATH اختیاری مسیر دلخواه برای WebSocket proxy. پیش‌فرض /.
URL اختیاری آدرس صفحه camouflage. می‌تونی nginx یا آدرس یه سایت مثل https://google.com بدی.
ساخته شده توسط گروه نوا پروکسی
